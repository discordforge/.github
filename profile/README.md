## About DiscordForge

DiscordForge is a curated directory for Discord server administrators and bot developers. Our team hand-reviews every submission to keep the platform free of spam and scams.

*   **For Server Admins**: Discover verified bots to manage, secure, and grow your communities.
*   **For Bot Developers**: List your bots, earn the verification badge, and reach thousands of early adopters.

## Features & Ecosystem

Built-in platform features:
*   **Manual Verification**: We review every bot and server before listing.
*   **Ranking & Voting System**: Community-driven visibility for the best tools.
*   **Pioneer Badge**: Exclusive recognition for early adopters.

### Tier System
Listing options for developers:
*   **Free**: Basic directory listing.
*   **Premium ($1.99/mo)**: Enhanced visibility and access to an analytics dashboard.
*   **Forge**: Top tier for maximum marketplace exposure.

## Roadmap

Coming soon:
*   **Official NPM Package**: We are building a `@discordforge-sdk` package for npm. This SDK lets developers interact with our API, automatically update bot statistics, and integrate platform features directly into their codebase.
